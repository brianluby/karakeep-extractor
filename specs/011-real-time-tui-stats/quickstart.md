# Quickstart: Real-time TUI Statistics

This feature is automatic when using the TUI mode.

## Enrichment
Run the enrichment command with the `--tui` flag.

```bash
karakeep enrich --tui
```

**Observation:**
Look below the progress bar. You will see live counters:
`Success: 120 | Failed: 2 | Skipped: 5`

## Extraction
Run the extraction command with the `--tui` flag.

```bash
karakeep extract --tui
```

**Observation:**
Similar counters will appear for processed bookmarks.
